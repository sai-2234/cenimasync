🌟 What is CenimaSync?

CenimaSync is a real-time watch-together platform that lets friends watch movies, chat, and play multiplayer games together — no matter the distance. Built in 2026 by Sai Chatre, it eliminates the "3… 2… 1… play!" countdown problem by syncing every pause, play, and rewind across all devices instantly via Firebase Firestore. Whether you're miles apart or just in different rooms, CenimaSync makes you feel like you're right there together.

✨ Core Features

🎬 Sync-Watch Rooms

Create a room, get a 5-letter code, and share it with anyone. Load any .mp4 or .webm video URL and press play — everyone sees the same frame at the same millisecond. No drift, no lag, no manual counting.

💬 Real-Time Chat

Instant messaging synced across all devices via Firestore. Themed bubbles, quick emoji reactions (❤️ 😂 🔥 😮 👏 🥹), timestamps, and customizable font sizes. Your conversations feel alive.

🎮 22+ Multiplayer Games

Board, arcade, party, word, and couples games — all synced in real-time. Invite friends directly from inside any game. Turn-based controls ensure only the right player can move on their turn.

🔗 Smart Invite System

When you start a game, everyone in the room sees a notification chip. Tap Invite next to any friend's name — they get a big popup to accept or decline instantly.

🎨 Customization

Tap the ⚙️ gear icon to access: Theme (Dark/Light), Accent Color (Violet, Cyan, Pink, Lime), Chat Bubble Style, Font Size, Compact Mode, Reduce Motion, Sound Effects, and Notifications. All settings persist across sessions.

⚙️ Technical Architecture

🔥 Firebase Firestore

Real-time database powering all sync. Rooms, peers, chat, games, and invites all live as Firestore subcollections with onSnapshot subscriptions for instant updates.

🛠 Tech Stack

Frontend: Vanilla HTML/CSS/JS optimized for mobile Chrome
Styling: CSS custom properties + glass-morphism + fluid blur
Animations: Pure CSS keyframes with cubic-bezier easing
Real-time: Firebase Firestore (onSnapshot)
Storage: Firebase Firestore + localStorage for settings

🔄 Sync Mechanism

Every user gets a stable peerId (stored in localStorage). Video state updates via updateDoc() → triggers onSnapshot() on all devices. Drift detection: if local video is >0.8s out of sync, it auto-corrects silently.

🚀 How to Use

1
Create Room — Enter your name, tap Create Room, get a 5-letter code
2
Share Code — Send the code to friends. They tap Join Room and enter it
3
Load Video — Paste any .mp4/.webm URL in the Watch tab and tap Load
4
Press Play — Syncs for everyone instantly. Pause/seek also syncs!
5
Chat — Tap the Chat tab to message in real-time with emoji reactions
6
Play Games — Tap Games tab, pick any game, invite friends from inside it
